Le système utilise la formule de Haversine pour calculer la distance orthodromique entre le match (lat2, lon2) et votre zone (lat1, lon1).

Si distance <= rayon_choisi, un webhook déclenche l'API Firebase Cloud Messaging pour vous envoyer une push notification instantanée.
